Installazione

$ platform
> Termux, Linux, WSL, Android (via Termux)

$ install termux
pkg update
pkg install git python bash coreutils
git clone https://github.com/HighKali/Ghost_Ops_Unit.git
cd Ghost_Ops_Unit
bash ghostos_autoload.sh

$ install linux
sudo apt update
sudo apt install git python3 bash coreutils
git clone https://github.com/HighKali/Ghost_Ops_Unit.git
cd Ghost_Ops_Unit
bash ghostos_autoload.sh

$ first_run
ghost
